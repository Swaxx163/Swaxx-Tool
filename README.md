## 🔥 Overview

Pappa Swaxx Tools is a multifunctional hacking toolkit with a cyberpunk
/ Mr. Robot aesthetic. It includes powerful network utilities such as
UDP/HTTP flood attacks, IP intelligence with VPN detection, port
scanning, subdomain enumeration, WHOIS lookups, traceroute, and more.

**Key feature:** Every user **must** enter their Discord ID before
accessing the tool. The tool logs all usage (including public IP, local
IP, system info, and VPN status) to a Discord webhook, making it easy to
track who is running the tool and from where.

------------------------------------------------------------------------

## 🧰 Features

-   💀 **UDP Flood** -- Adjustable strength (1--10) with bandwidth
    estimation and safety warnings.
-   🔥 **HTTP/HTTPS Flood** -- Multi‑threaded GET/POST attacks.
-   🌍 **IP Intelligence** -- Geolocation, reverse DNS, and VPN/proxy
    detection using ip-api.com.
-   🔍 **IP Finder** -- Show your local and public IP addresses.
-   🔎 **Port Scanner** -- Fast TCP port scanning with service
    detection.
-   📶 **Ping Utility** -- Latency, jitter, and RTT statistics.
-   🌐 **DNS Resolver** -- Resolve A records (IPv4).
-   📋 **WHOIS Lookup** -- Fetch domain registration data via raw WHOIS.
-   🔍 **Subdomain Scanner** -- Brute‑force common subdomains.
-   🌐 **Network Scanner** -- Scan your local subnet for active devices.
-   🔀 **Traceroute** -- Show the network route to a target.

All functions are wrapped in error handling and come with a
**Matrix‑style progress bar** and **colourful hacker‑themed output**.

------------------------------------------------------------------------

## 📦 Installation

### 1. Clone or download the script

Save the script as `tool.py` (or whatever you like).

### 2. Install required dependencies

The tool relies on a few Python libraries. Install them with pip:

\`\`\`bash pip install requests
